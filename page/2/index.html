  • 日期:2026-01-16
  • 周次:第 4 周

学习目标

今天是第 4 周的第一天,你应该能做到:

  • 理解漏洞扫描的合法边界:能清楚说明授权范围、扫描窗口、合规要求
  • 完成 OpenVAS/Nessus Essentials 安装:能在 Windows 或 Linux 环境下完成部署
  • 执行一次标准扫描:针对授权目标(本机/VM)执行基础漏洞扫描
  • 解读扫描报告:能识别高/中/低危漏洞,理解 CVSS 评分体系

核心原则:漏洞扫描器是强大的工具,必须在合法授权范围内使用。未经授权扫描他人系统属于违法行为。
