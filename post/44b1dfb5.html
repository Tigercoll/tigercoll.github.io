一、PowerShell 基础语法与对象管道

1.1 什么是 PowerShell？

  • Windows 自带的命令行自动化工具，支持“对象管道”，比传统 cmd 强大很多
  • 语法类似 C#/Python，支持变量、函数、条件、循环、模块
  • 适合做批量运维、日志分析、自动化脚本

1.2 管道与重定向的区别

概念 作用 示例 结果
管道 ` ` 把前一命令的“对象”传给后一命令 `Get-Process
重定向 > 把输出写到文件 Get-Process > procs.txt 文件里是纯文本
追加 >> 追加到文件末尾 echo hello >> log.txt 文件内容不会被覆盖

1.3 PowerShell 与 Bash 最大差异

  • PowerShell 管道传递的是“对象”，不是纯文本
  • 可以直接用 Select-Object、Where-Object、Group-Object 等命令处理结构化数据

五、巩固练习与评估标准

巩固练习

  1. 用 PowerShell 写一个脚本，统计每小时的登录次数（按时间分组）
  2. 写一个脚本，找出 1 小时内操作次数最多的用户
  3. 用正则提取所有“fail”操作的时间和用户名
  4. 写一个函数，批量处理多个日志文件，统计总的“fail”次数
  5. 用 try/catch 包裹文件读取，遇到不存在的文件输出“文件缺失”

评估标准

  • ✅ 能用管道和 Group-Object 做分组统计
  • ✅ 能用 Export-Csv 导出结果
  • ✅ 能写出带参数的函数
  • ✅ 能用 try/catch 做错误处理
  • ✅ 能用正则提取日志字段
